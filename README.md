# Planet Renderer

This is a simple C++ planet renderer I wrote when I was 16 years old.
It uses the OpenGL API for 3D rendering.

## Controls

* WASD - Move horizontally
* Space - Move up
* Shift - Move down
* Control - Move fast
* Alt - Move faster
* Escape - Release mouse
* Left mouse button - Grab mouse
